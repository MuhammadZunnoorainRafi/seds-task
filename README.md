# Task Web Appliction

This is a simple full-stack web application for managing tasks.

# Run Locally

### Step # 1

Accept the github private repo invite that I have given you.

### Step # 2

For Installing all dependencies (Do this from the root of this application)

```bash
  npm run mount
```

### Step # 3

➡️ **IMPORTANT:** Create new ".env" file in the root of the application and Copy & paste all envs from the bottom of file "README.md" (which is present in the root and are commented) to the ".env"

### Step # 4

Start the server (Do this from the root of this application)

```bash
  npm run dev
```

# Libraries Used

- **Mongoose** (ODM).
- **Json Web Token** - For securily transmitting information and for authorization.
- **Zod** - For schema and form validations.
- **React Hook Form** - For managing forms.
- **Radix** - For styling.
- **Redux Toolkit** - For state management.
- **React Query** - For simplifying data fetching and caching.
- **React Router Dom** - For handling routing

# Bonus Features
